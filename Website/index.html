Ship a static site to Amazon S3 with one command.

CloudDeploy is a lightweight Python automation script that creates your bucket, configures public hosting, uploads every file with the correct MIME type, and hands you a live URL — no console clicking required.

Why CloudDeploy

  
Everything the manual console click-through does — automated.


  

    

      
🪣

      
Bucket automation

      
Creates the S3 bucket only if it doesn't already exist, so the script is safe to re-run anytime.

    

    

      
📤

      
Recursive upload

      
Walks your website folder and uploads every file, preserving nested folder structure automatically.

    

    

      
🧠

      
Smart MIME detection

      
Detects Content-Type per file so HTML renders, CSS applies, and images display — not download prompts.

    

    

      
🌐

      
Hosting configuration

      
Enables static website hosting and sets index/error documents through the SDK, not the console.

    

    

      
🔐

      
Least-privilege policy

      
Attaches a public read-only bucket policy — visitors can view files, nobody can write or delete.

    

    

      
📜

      
Logs & error handling

      
Every step is logged to console and file, with clear exceptions instead of raw AWS tracebacks.

    

  







  
Project workflow

  
From local folder to live URL


  

    


    

      
01

      
Run the script

      
Developer runs python upload_website.py from the project root.

    

    

      
02

      
boto3 connects

      
The SDK authenticates using your local AWS CLI credentials.

    

    

      
03

      
Bucket created

      
S3 bucket is created (or reused) and public access is configured.

    

    

      
04

      
Files uploaded

      
Every website file is uploaded with the correct content type.

    

    

      
05

      
Hosting enabled

      
Static website hosting is switched on for the bucket.

    

    

      
06

      
URL returned

      
The live website endpoint is printed straight to your terminal.

    

  







  

    

      
About the stack

      
Built entirely on AWS Free Tier

      

        This project intentionally uses only services within the AWS Free Tier: Amazon S3 for
        storage and hosting, IAM for secure access, and the AWS CLI / boto3 SDK for automation.
        No servers, no containers, no ongoing compute cost — just object storage configured
        to behave like a web server.
      

      
    
        
  • S3 Object storage & static website hosting
    • 
        
  • IAM Scoped user credentials for the script
    • 
        
  • CLI Local credential configuration
    • 
        
  • boto3 Python SDK driving every AWS call

Clone the repo, drop your HTML/CSS/JS into website/, set your bucket name, and run the script.

cd StaticWebsiteAutomation
pip install -r requirements.txt
python upload_website.py
